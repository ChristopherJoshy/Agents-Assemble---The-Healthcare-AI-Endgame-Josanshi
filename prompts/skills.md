# Skills

Use these as individual entries in the Prompt Opinion `Skills` UI.

Each skill below is formatted as:

- `Name`
- `Description`

## Recommended Skills For Josanshi

### 1. Postpartum Risk Review

**Name**

`Postpartum Risk Review`

**Description**

Analyzes maternal and postpartum risk using Josanshi MCP tools, including hemorrhage, cardiovascular, mental-health, transition, and coverage-cliff signals. Best used when another agent needs a grounded postpartum risk summary with urgency and next steps.

### 2. Maternal Mental Health Review

**Name**

`Maternal Mental Health Review`

**Description**

Interprets postpartum depression screening results, including PHQ-9 and Edinburgh workflows, and identifies urgent self-harm escalation signals. Use when another agent needs safe interpretation of maternal mental-health screening results.

### 3. Pregnancy And Lactation Medication Safety

**Name**

`Pregnancy And Lactation Medication Safety`

**Description**

Reviews whether a medication is safe in pregnancy and breastfeeding using the Josanshi MCP medication-safety workflow. Includes pregnancy-risk framing, Hale lactation categories, and FAERS-aware safety interpretation.

### 4. Maternal Vaccine Timing

**Name**

`Maternal Vaccine Timing`

**Description**

Checks whether maternal vaccines are due, documented, or overdue based on available immunization data and CDC/ACIP-aligned timing logic. Use for Tdap, influenza, COVID, and related maternal immunization questions.

### 5. Postpartum Coverage Cliff Tracking

**Name**

`Postpartum Coverage Cliff Tracking`

**Description**

Calculates days until postpartum Medicaid coverage expiry, identifies outstanding screenings before coverage ends, and highlights state extension context. Use when another agent needs insurance-risk and follow-up-gap analysis.

### 6. Maternal Equity Disparity Analysis

**Name**

`Maternal Equity Disparity Analysis`

**Description**

Analyzes maternal-care delivery disparities across patient groups, including referral delays, postpartum visit completion, care-gap rates, and coverage-loss patterns. Use for panel-level operational and equity review.

### 7. Maternal Clinical Data Summary

**Name**

`Maternal Clinical Data Summary`

**Description**

Retrieves and summarizes maternal patient demographics, vitals, medications, conditions, and anemia-related findings from Josanshi MCP tools. Use when another agent needs a concise, grounded clinical snapshot before deeper reasoning.

### 8. Maternal Safety Escalation Review

**Name**

`Maternal Safety Escalation Review`

**Description**

Determines whether maternal findings require routine follow-up, urgent review, or immediate escalation. Best used when another agent wants a conservative safety check before presenting a clinician-facing answer.

## Best Minimal Set

If you only want the highest-value entries, add these five:

1. `Postpartum Risk Review`
2. `Maternal Mental Health Review`
3. `Pregnancy And Lactation Medication Safety`
4. `Postpartum Coverage Cliff Tracking`
5. `Maternal Equity Disparity Analysis`

## Suggested Default Set

If the UI allows several skills, use all eight above.
